# PhonePe_Business_Dashboard
🧩 Overview
The PhonePe Business Dashboard is a data analytics and visualization project built using Power BI, SQL, Python, and Excel. It analyzes one year of transaction data from the PhonePe digital payment ecosystem to uncover insights about user behavior, merchant performance, and regional adoption trends.

This dashboard provides a unified view of key business metrics, enabling stakeholders to make data‑driven decisions and monitor growth across India’s digital payment landscape.

🚀 Key Features
Interactive Power BI Dashboard with dynamic filters and drill‑down capabilities

Transaction Analytics: Total transactions, value, success rate, and growth trends

User Insights: Unique users, age‑segment contribution, and engagement patterns

Service Analysis: Loans, money transfers, insurance, and recharge performance

Top Users Visualization: Ranking by transaction value

Temporal Trends: Monthly and weekday/weekend usage comparison

Performance Metrics: Success rate of 96.15% and 8.9% growth in transaction volume

🛠️ Tech Stack
Tool	Purpose
SQL	Data extraction, cleaning, and aggregation
Python	Data preprocessing, trend analysis, and validation
Excel	Data validation and pivot summaries
Power BI	Visualization, modeling, and dashboard creation


📈 Insights
